## What you're deciding
How you want to provide the data we need for each of the 7 metrics approved in Stage 4. You have two options and can pick whichever is easier for you — the output of Stage 5 is the same either way.

- **Answer inline** — I'll create a follow-up file listing the specific questions per metric, and you fill in the answers directly in the `[Answer]:` lines. Best when the numbers are in your head, in a dashboard you can quickly pull, or when you want full control over each value.
- **Upload documents** — I'll create an `uploads/` subfolder inside `05-data-collection/`, and you drop in one or more plain-text or markdown files (exports, memos, dashboard snapshots exported to text, or any notes containing the data). I'll extract the data points from what you provide, note anything I couldn't find, and ask targeted follow-up questions only where needed. Best when the information already exists in a document and you don't want to retype it.

You can mix approaches — start with uploads and fill any gaps inline afterward.
